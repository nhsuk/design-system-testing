New component Tag

Background

Backlog issue 174 (GitHub)

The design and code is based on the GOV.UK Design System tag component. It has been tested by various NHS services, including the coronavirus testing service and the NHS App.

When to use a tag

Use the tag component when it’s possible for something to have more than one status and it’s useful for the user to know about that status. For example, you can use a tag to show whether an item in a task list has been ‘completed’.
