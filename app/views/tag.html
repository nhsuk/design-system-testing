Component Tag

Background

A tag component has been proposed in the backlog, Backlog issue 174. The GOV.UK Design System tag component was recently released.

When to use this component

Use the tag component to indicate the status of something, such as an item on a task list.

Using coloured tags

Class name Tag
nhsuk-tag Submitted
nhsuk-tag--white Started
nhsuk-tag--grey Not started
nhsuk-tag--green New
nhsuk-tag--aqua-green Active
nhsuk-tag--blue Pending
nhsuk-tag--purple Received
nhsuk-tag--pink Sent
nhsuk-tag--red Rejected
nhsuk-tag--orange Declined
nhsuk-tag--yellow Delayed

Accessibility

All contrasts between text and background meet level AAA of WCAG 2.0. Read more about accessibility and colour.
