Proposal to update guidance Review date

Overview

Proposing to rewrite the review date component as a design pattern, remove the review date code and build the pattern using existing styles and components.

Styles

The review date could be built using these styles:

  • nhsuk-body-s
  • nhsuk-u-secondary-text-color
  • spacing overrides, for example nhsuk-u-margin-top-7

Page last reviewed: 12 February 2017
Next review due: 1 February 2020

Updated: June 2019

Components

Doesn't need a component.

Pattern

The component guidance would be rewritten as a design pattern: "Reassure users that your information is up-to-date".
